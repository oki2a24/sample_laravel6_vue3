# sample_laravel6_vue3
Laravel6 で Vue.js 3 を使うサンプルです。

## リスト
- コンテナの exim4 MTA からメール送信可能にした。
- Vue.js 3 SFC. Bootstrap4 のページネーションを追加した。
- ファイルアップロードの API とユニットテストを追加した。
- Vue.js 3 SFC. デイトピッカー flatpickr を追加した。
- Vue.js 3 SFC. Dropzone.js を追加した。
- Vue.js 3 SFC. Select2 を追加した。
- Vue.js 3 SFC. Bootstrap4 のモーダルを追加した。
- Laravel6 環境で Vue.js 3, Vue Router を追加した。
- Laravel6 ユニットテストで Docker 環境変数で上書き可能にする。
- VS Code Remote Container で Veture 拡張機能を使えるようにした。
- Larvel 6 に入れた Vue.js 3 の環境で、 ESLint と Prettier を使えるようにした。
